## üÜò Getting Help

### Within Ken
```
You: help
You: what can you do?
You: show all commands
```

### Check Your Data
```
You: /meta                          # Storage overview
You: /mem list                      # Your memories
You: /reminder list                 # Your reminders
```

### Debug Mode
```
You: /debug on                      # See detailed progress
```
