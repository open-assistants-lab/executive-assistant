Approach Options

A

Unified "Agent Resources" panel

Merge Skills + Subagents into one panel with a shared design system. Skills and agents shown in a single list with type indicators, cross-references. Subagent editor surfaces skills at top level, not under Advanced.

Pros

  • One consistent UI
  • Clear skill↔agent relationships
  • Less tab switching

Cons

  • More complex layout
  • Harder to navigate with many items
B

Keep tabs, consistent redesign (Recommended)

Both panels use identical list tile components. Subagent dialog restructured: skills moved out of Advanced into a prominent section. Skills tab shows usage count (which subagents use this). Cross-linking: tap a skill chip on a subagent tile → highlights that skill.

Pros

  • Minimal structural change
  • Clear separation of concerns
  • Skills visible at a glance

Cons

  • Still need to switch tabs
C

Master-detail with inspector

Left: collapsed list of all skills + subagents. Right: detail panel for the selected item. Selecting a subagent shows its config including assigned skills inline. Selecting a skill shows its content and which subagents use it.

Pros

  • No tab switching
  • Full detail without dialogs
  • Best cross-referencing

Cons

  • Major layout redesign
  • Very different from current UI
  • Might feel cramped in a side panel
